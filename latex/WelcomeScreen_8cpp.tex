\doxysubsection{Detailed Description}
Implements the \mbox{\hyperlink{classWelcomeScreen}{Welcome\+Screen}} widget which displays a personalized welcome message, user heart rate statistics, and a historical chart. 

The \mbox{\hyperlink{classWelcomeScreen}{Welcome\+Screen}} class reads the user\textquotesingle{}s heart rate data from a CSV file (\char`\"{}userdata.\+csv\char`\"{}), calculates statistics such as the average and latest heart rate, and determines the user\textquotesingle{}s risk level. It also provides navigation buttons, including one to display tailored health tips via the \mbox{\hyperlink{classTipsForUser}{Tips\+For\+User}} widget.

\begin{DoxyNote}{Note}
This widget is designed to be used within a QStacked\+Widget for screen navigation.
\end{DoxyNote}
