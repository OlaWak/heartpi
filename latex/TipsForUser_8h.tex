\doxysubsection{Detailed Description}
Declaration of the \mbox{\hyperlink{classTipsForUser}{Tips\+For\+User}} widget. 

This file declares the \mbox{\hyperlink{classTipsForUser}{Tips\+For\+User}} class, which displays curated health tips to help users improve their heart health. The tips are tailored based on the user\textquotesingle{}s risk level and are presented within a custom-\/styled widget. The widget also supports navigation back to a previous screen.

\begin{DoxyNote}{Note}
The widget overrides the paint\+Event() function to render a custom linear gradient background.
