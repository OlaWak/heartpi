\doxysubsection{Detailed Description}
Implements the \mbox{\hyperlink{classTipsForUser}{Tips\+For\+User}} widget that displays curated health tips for a healthier heart. 

This file defines the constructor and paint\+Event() function for the \mbox{\hyperlink{classTipsForUser}{Tips\+For\+User}} widget. The widget displays various health tips organized into categories such as activity, nutrition, monitoring, and sleep \& stress. Tips are tailored based on the user\textquotesingle{}s risk level (\char`\"{}\+High\char`\"{}, \char`\"{}\+Moderate\char`\"{}, or other), and a scrollable layout is used to present the content. A back button is provided to navigate to the previous screen.
