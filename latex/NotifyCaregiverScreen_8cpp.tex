\doxysubsection{Detailed Description}
Implements the \mbox{\hyperlink{classNotifyCaregiverScreen}{Notify\+Caregiver\+Screen}} widget for sending alert emails to caregivers. 

This file defines the \mbox{\hyperlink{classNotifyCaregiverScreen}{Notify\+Caregiver\+Screen}} class methods which allow a user to select their account, enter a password, and provide a recipient email to send a Heart\+Pi alert. The screen loads available accounts from a CSV file, verifies credentials, fetches heart rate data, computes risk levels, and composes an alert email using the \mbox{\hyperlink{classEmailSender}{Email\+Sender}} module.

\begin{DoxyNote}{Note}
The CSV file \char`\"{}userdata.\+csv\char`\"{} is used both for registration (2-\/column rows) and heart rate data (3-\/column rows).
\end{DoxyNote}
