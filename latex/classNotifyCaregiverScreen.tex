\doxysubsection{Detailed Description}
A widget for sending alert emails to caregivers. 

The \mbox{\hyperlink{classNotifyCaregiverScreen}{Notify\+Caregiver\+Screen}} class allows a user to send a Heart\+Pi alert email by selecting their account, entering a password, and specifying a recipient email address. It loads available accounts from a CSV file and verifies the provided credentials.
