\doxysubsection{Detailed Description}
A widget for user login to view previous results. 

The \mbox{\hyperlink{classResultsLoginScreen}{Results\+Login\+Screen}} class provides an interface for users to log in by selecting an account and entering a password. It verifies the credentials and emits a signal upon a successful login.
