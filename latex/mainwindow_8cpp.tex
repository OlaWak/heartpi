\doxysubsection{Detailed Description}
Implements the \mbox{\hyperlink{classMainWindow}{Main\+Window}} class which serves as the central hub for the Heart\+Pi application. 

The \mbox{\hyperlink{classMainWindow}{Main\+Window}} sets up the main menu with an animated background and provides navigation between various screens\+: \mbox{\hyperlink{classSurveyScreen}{Survey\+Screen}} for user registration/login, \mbox{\hyperlink{classSurveyFormScreen}{Survey\+Form\+Screen}} for survey input, \mbox{\hyperlink{classHeartHealthScreen}{Heart\+Health\+Screen}} for displaying heart health statistics, \mbox{\hyperlink{classResultsLoginScreen}{Results\+Login\+Screen}} for viewing past results, \mbox{\hyperlink{classWelcomeScreen}{Welcome\+Screen}} for personalized greetings, and \mbox{\hyperlink{classNotifyCaregiverScreen}{Notify\+Caregiver\+Screen}} for sending alerts.

\begin{DoxyNote}{Note}
The application uses a QStacked\+Widget to manage screen transitions.
