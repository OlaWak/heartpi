\doxysubsection{Detailed Description}
A widget with a custom background and animated heartbeat wave. 

The \mbox{\hyperlink{classCustomBackgroundWidget}{Custom\+Background\+Widget}} class inherits from QWidget and provides a custom-\/painted background featuring a linear gradient. When animation is enabled, it displays a heartbeat wave animation. It emits a resized() signal whenever the widget is resized.
