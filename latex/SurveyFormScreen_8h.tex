Declaration of the \mbox{\hyperlink{classSurveyFormScreen}{Survey\+Form\+Screen}} widget. 

This file declares the \mbox{\hyperlink{classSurveyFormScreen}{Survey\+Form\+Screen}} class, which provides a user interface for collecting heart health survey data. It displays a series of questions using combo boxes, collects the responses, and packages the data into a \mbox{\hyperlink{classFamilyHealth}{Family\+Health}} object. Once the survey is submitted, a signal is emitted containing the collected data.

\begin{DoxyNote}{Note}
This widget is intended to be used within a QStacked\+Widget for screen navigation.
