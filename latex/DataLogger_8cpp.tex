\doxysubsection{Detailed Description}
Implements the \mbox{\hyperlink{classDataLogger}{Data\+Logger}} class methods for logging sensor data. 

This file contains the implementation of the \mbox{\hyperlink{classDataLogger}{Data\+Logger}} class, which logs sensor data (e.\+g., heart rate, blood pressure, cholesterol, ECG) to a file. It handles file operations and error reporting using the \mbox{\hyperlink{classErrorHandling}{Error\+Handling}} class.

\begin{DoxyNote}{Note}
The log file is opened in append mode. If the file is empty, a header row is written.
\end{DoxyNote}
