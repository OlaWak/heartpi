Declaration of the \mbox{\hyperlink{classFamilyHealth}{Family\+Health}} class for collecting personal and family health data. 

This file defines the \mbox{\hyperlink{classFamilyHealth}{Family\+Health}} class, which stores a user\textquotesingle{}s personal health details such as age group, gender, sleep hours, exercise frequency, diet type, and smoking status. It also maintains a record of family disease history using dynamic vectors.
