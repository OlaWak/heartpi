\doxysubsection{Detailed Description}
Logs sensor data to a CSV file. 

The \mbox{\hyperlink{classDataLogger}{Data\+Logger}} class handles file operations for logging sensor data. It writes the data along with timestamps to a CSV file, ensuring that data is appended correctly and the file is properly closed.
