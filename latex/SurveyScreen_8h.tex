\doxysubsection{Detailed Description}
Declaration of the \mbox{\hyperlink{classSurveyScreen}{Survey\+Screen}} widget. 

This header file declares the \mbox{\hyperlink{classSurveyScreen}{Survey\+Screen}} class, which provides an interface for users to register or log in before taking the heart health survey. The widget includes fields for username and password along with buttons to start the survey or navigate back. On successful registration, it emits a signal indicating that the survey login was successful.

\begin{DoxyNote}{Note}
This widget is designed to be used within a QStacked\+Widget for navigation between screens.
\end{DoxyNote}
