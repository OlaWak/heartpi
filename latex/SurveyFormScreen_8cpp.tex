\doxysubsection{Detailed Description}
Implements the \mbox{\hyperlink{classSurveyFormScreen}{Survey\+Form\+Screen}} widget for collecting heart health survey data. 

This file defines the \mbox{\hyperlink{classSurveyFormScreen}{Survey\+Form\+Screen}} class methods. The widget displays a survey form with various questions regarding the user\textquotesingle{}s health, collects the responses, and then emits a signal with the collected \mbox{\hyperlink{classFamilyHealth}{Family\+Health}} data. Upon submission, it transitions to the \mbox{\hyperlink{classHeartHealthScreen}{Heart\+Health\+Screen}} to display results.

\begin{DoxyNote}{Note}
The survey questions include age group, gender, sleep hours, exercise frequency, family history, diet type, and smoking status.
\end{DoxyNote}
