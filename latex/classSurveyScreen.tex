\doxysubsection{Detailed Description}
A widget for user registration/login for the heart health survey. 

The \mbox{\hyperlink{classSurveyScreen}{Survey\+Screen}} class provides input fields for a username and password, along with buttons to start the survey or go back to the main screen. It validates the inputs, saves user data to a CSV file, and emits a signal upon successful registration.
