\doxysubsection{Detailed Description}
Implements the \mbox{\hyperlink{classHeartHealthScreen}{Heart\+Health\+Screen}} widget for displaying heart health results and live charts. 

This file defines the \mbox{\hyperlink{classHeartHealthScreen}{Heart\+Health\+Screen}} class methods, which provide a user interface to display heart health assessments, live heart rate charts, and historical data. The widget uses Qt Widgets and Qt Charts, integrates with the Calculations and \mbox{\hyperlink{classRandomNumberGenerator}{Random\+Number\+Generator}} modules, and provides visual and audio feedback based on risk levels.

\begin{DoxyNote}{Note}
