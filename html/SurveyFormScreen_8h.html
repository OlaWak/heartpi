Detailed Description

Declaration of the SurveyFormScreen widget.

This file declares the SurveyFormScreen class, which provides a user interface for collecting heart health survey data. It displays a series of questions using combo boxes, collects the responses, and packages the data into a FamilyHealth object. Once the survey is submitted, a signal is emitted containing the collected data.

Note
This widget is intended to be used within a QStackedWidget for screen navigation.
