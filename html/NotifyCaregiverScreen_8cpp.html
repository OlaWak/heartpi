Detailed Description

Implements the NotifyCaregiverScreen widget for sending alert emails to caregivers.

This file defines the NotifyCaregiverScreen class methods which allow a user to select their account, enter a password, and provide a recipient email to send a HeartPi alert. The screen loads available accounts from a CSV file, verifies credentials, fetches heart rate data, computes risk levels, and composes an alert email using the EmailSender module.

Note
The CSV file "userdata.csv" is used both for registration (2-column rows) and heart rate data (3-column rows).
