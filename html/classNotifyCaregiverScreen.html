Detailed Description

A widget for sending alert emails to caregivers.

The NotifyCaregiverScreen class allows a user to send a HeartPi alert email by selecting their account, entering a password, and specifying a recipient email address. It loads available accounts from a CSV file and verifies the provided credentials.

void NotifyCaregiverScreen::attemptSendAlert ( )
privateslot

Attempts to send an alert email to the caregiver.

Validates user input, verifies credentials, and then fetches heart rate data from "userdata.csv". Computes average and latest heart rate, determines the risk level, composes the email subject and body, and sends the email using the EmailSender class. Displays appropriate messages based on success or failure.
