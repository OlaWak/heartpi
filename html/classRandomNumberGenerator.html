Detailed Description

Generates random numbers within a specified range.

This class uses the Mersenne Twister random number engine (std::mt19937) along with a uniform real distribution to generate random numbers. It is particularly useful for simulating sensor readings.

Note
The random engine is seeded using std::random_device to ensure a different sequence of random numbers each time the program runs.
