Detailed Description

Implements the HeartHealthScreen widget for displaying heart health results and live charts.

This file defines the HeartHealthScreen class methods, which provide a user interface to display heart health assessments, live heart rate charts, and historical data. The widget uses Qt Widgets and Qt Charts, integrates with the Calculations and RandomNumberGenerator modules, and provides visual and audio feedback based on risk levels.

Note
The widget is designed to be used within a QStackedWidget for screen transitions.
