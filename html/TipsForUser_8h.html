Detailed Description

Declaration of the TipsForUser widget.

This file declares the TipsForUser class, which displays curated health tips to help users improve their heart health. The tips are tailored based on the user's risk level and are presented within a custom-styled widget. The widget also supports navigation back to a previous screen.

Note
The widget overrides the paintEvent() function to render a custom linear gradient background.
